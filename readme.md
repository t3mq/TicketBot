# Ticket Bot

Ticket Bot is a discord ticket bot with buttons made with Discord.js v14

## How to install ?

You need to have Node.JS 16+
``````bash
git clone https://github.com/Xiaotoxdev/TicketBot.git
cd ticket-bot
npm i
``````

## How to config ?

```json
//config.json
{
    "token": "",
    "parent": "",
    "roleStaffId": "",
    "logChannel": ""
}
```
## How to start ?
```bash
node index.js # To start the ticket-bot
```
